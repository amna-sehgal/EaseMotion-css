# Textarea Resize Label Fix (Issue #14073)\n\n## What does this do?\nShows correct min-height and max-height on textarea to prevent resize handle from overlapping the label.\n\n## How is it used?\n```html\n<div class="ease-field">\n  <label class="ease-field-label">Message</label>\n  <textarea class="ease-textarea"></textarea>\n</div>\n```\n\n## Why is it useful?\nAdding `max-height` prevents users from dragging the textarea resize handle upward to overlap the label, maintaining a clean form layout.
